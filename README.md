## xditx32 Personal Blog 

### How to Install 

#### Prerequisites:

```
ruby dk.rb init
ruby dk.rb install
gem install bundler
```

#### Install local:

```
bundle install
```

#### Run local:

```
jekyll serve --config _config-dev.yml
```
